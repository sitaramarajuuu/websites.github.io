# 🦕 Dino Run!

A browser-based clone of Chrome's offline dinosaur game. Fully self-contained in a single HTML file — no dependencies, no build step.

## 🎮 How to Play

- **Jump**: `Space` / `↑ Arrow` / Tap screen
- **Duck**: `↓ Arrow` / Duck button (mobile)
- Avoid cacti and pterodactyls
- Score increases over time; speed ramps up as you go
- Night mode toggles every 700 points

## 🚀 Deploy to GitHub Pages

1. Fork or clone this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your game will be live at `https://<username>.github.io/<repo-name>/`

## 📁 Files

| File | Description |
|------|-------------|
| `index.html` | The entire game — HTML, CSS, JS, and pixel-art sprites all in one file |

## Features

- Pixel-art dino, cacti, and pterodactyl sprites drawn via Canvas
- Ducking mechanic with fast-fall in mid-air
- Day/night cycle
- High score saved in localStorage
- Mobile touch controls
- Increasing difficulty
